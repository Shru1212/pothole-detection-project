# Smart Pothole Detection System

A Python-based web application that detects and analyzes pothole severity from uploaded road images.

## Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, CSS
- Data Handling: CSV
- Tools: VS Code

## Features
- Upload pothole images through a web interface
- Analyze image to estimate pothole severity
- Display detection results on the web page
- Store results for reporting and analysis

## Project Structure
- app.py – Backend logic
- templates/ – HTML templates
- static/ – CSS and assets
- reports.csv – Detection output
- sample images – Test inputs

## How to Run
1. Install dependencies  
2. Run `python app.py`  
3. Open browser at `localhost`

## Learning Outcome
- Python backend development
- Web application structure
- Image-based analysis logic
